## Cat-tastic!

### Running the app

To use this app:

- `git clone` it
- [Generate an API key](https://thecatapi.com) for the Cat API
- Create an `.env.development.local` file
- Add the key like this: `VITE_CAT_API_KEY=YourApiKey`
- Do `npm install`
- Run `npm run dev`
- Enjoy it!

### Tech Stack

The app was created with:

- `React` + `Vite`
- `React Router`
- `Chakra UI`
- `React Query`
- Lots of cats :)
